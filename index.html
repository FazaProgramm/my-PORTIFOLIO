Meni Bilimlarim

Bular meni 8oy ichida o`rgangan narsalarim

HTML

HTML (HyperText Markup Language) — bu veb-sahifalarni yaratish uchun asosiy markup tili bo'lib, unda matn, tasvirlar, tugmalar va boshqa elementlarning tuzilishi aniqlanadi.

CSS

CSS (Cascading Style Sheets) — bu HTML elementlariga uslub (style) berish uchun ishlatiladigan til bo'lib, u orqali ranglar, shriftlar, joylashuvlar va boshqa dizayn jihatlari boshqariladi.

JavaScript

JavaScript (JS) — bu veb-sahifalarga interaktivlik qo'shish uchun ishlatiladigan dasturlash tili bo'lib, u orqali tugmalarni bosish, shakllarni yuborish, animatsiyalar va real vaqt ma'lumotlarini qayta ishlash kabi funksiyalarni amalga oshirish mumkin.

Bootstrap

Bootstrap — bu veb-sayt va veb-ilovalar yaratishni osonlashtiruvchi, oldindan tayyorlangan HTML, CSS va JavaScript komponentlar to'plamidan iborat ochiq kodli framework bo'lib, responsiv (moslashuvchan) dizayn yaratishda keng qo'llaniladi.

SASS

SASS (Syntactically Awesome Stylesheets) — bu CSS uchun kengaytma bo‘lib, u kodni soddalashtirish va takrorlanishni kamaytirish uchun o‘zgaruvchilar, funksiyalar, mixinlar va nesting kabi qulayliklarni taqdim etadi.

Git

Git — bu dasturiy ta'minot ishlab chiqishda versiya nazorati uchun ishlatiladigan tizim bo‘lib, u kod o‘zgarishlarini kuzatish, bir nechta dasturchilar bilan birgalikda ishlash va o‘tgan versiyalarni boshqarishni ta'minlaydi.
